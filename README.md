# Эксперимент "Энергия для деления"

У бота есть параметр, отвечающий за то, при каком количестве энергии происходит автоматическое деление. При мутации параметр немного изменяется.

Параметр не может быть больше 1000 или меньше 150

На записи чем больше параметр - тем бот краснее, чем меньше, тем бот желтее(как в режиме отрисовки энергии). Картинки в режиме отрисовки энергии для деления пишутся в папку "color"(вместо режима отрисовки цвета)
